# win9x_patch_boot
Windows 95/98/ME Bootable Patch

## Instructions:
1) Download the floppy image
2) Mount it to your virtual machine
3) Boot to floppy and reboot after done

## Note:
1) Currently there is just rloew's PATCHMEM included.
